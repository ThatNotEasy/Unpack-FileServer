### **Risk Your Life (RYL)** - MMORPG Overview

**Risk Your Life (RYL)** is a **Massively Multiplayer Online Role-Playing Game (MMORPG)** developed by **Youxiland** and **Gamasoft**. It was originally released in the early 2000s and became popular in several regions, particularly in Southeast Asia.

- **[Eternal](https://www.ryl2eternal.com/)**: Files for Eternal server version (scripts, DLLs, zones).
- **[GempakZ](https://ryl2gempakz.com/)**: Files for GempakZ server version.
- **Unity/**: Files for Unity server version. (Discontinue)
- **[Mapley](http://ryl2maplex.com/)**: Files for Unity server version.
- **[International](http://malaya.sytes.net/)**: Files for Unity server version.
- **[EndlessWaltZ](http://www.ryl2endless.com/)**: Files for GempakZ server version.

#### **Private Servers**: 
Since the official servers were shut down, private servers (like the ones mentioned in the repository) have become popular for keeping the game alive. These servers can have different rules, custom features, and content that is sometimes unique to that server version.
